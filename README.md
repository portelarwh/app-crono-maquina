# Crono Máquina

Cronômetro web para controle de tempo de **uma máquina por vez**.

## Estrutura

```
├── index.html
├── styles.css
├── app.js
├── .gitignore
└── assets/
```

## Funcionalidades atuais

- Campo de texto livre para identificar a máquina/contexto.
- Cronômetro único com ações: iniciar, pausar e zerar.
- Persistência local (LocalStorage) para manter estado ao recarregar.
- Tema escuro fixo (sem alternância claro/escuro).

## Como usar

Abra `index.html` diretamente no browser ou sirva via GitHub Pages.
